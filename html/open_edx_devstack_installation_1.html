Please install the master branch of the Open edX developer stack. There are two versions of it:

  • a newer Docker version (the README of its GitHub repo has pretty explicit, simple, well-tested steps on how it's setup)
  • an older Vagrant devstack (to get it up and running, please follow OpenCraft custom instructions, edX simple instruction or complete explanations)

From the experience of the past newcomers it seems like the Docker devstack is easier to get started with. Although, for some tasks (e.g. McKinsey) you may need a Vagrant devstack. If you're not sure which one to choose, please ask the reviewer of your on-boarding JIRA task.
